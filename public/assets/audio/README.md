# Reach for the Sky Audio Assets

`reach-ui-cues.ogg` is a generated OGG/Opus Howler sprite for short UI and simulation cues.
Tone.js remains the procedural fallback when the sample is unavailable, loading, muted, or blocked by browser audio policy.

Sprite map in `src/audio/audioEngine.ts`:

- `build`: construction confirm
- `rent`: revenue collection / visit spend
- `warning`: failed contract, rent leak, weather stress
- `elevator`: transit and visitor movement accent
- `milestone`: act completion and victory

The file was generated locally from simple synthesized tones and contains no third-party material.
